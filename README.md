# Knapsack

Uses the Knapsack algorithm to make get the best outcome of items with set values and weights.

This code uses the following table.


| Item | Value | Size |
| ---- | ----- | ---- |
|  1   |  1    |  1   |
|  2   |  2    |  3   |
|  3   |  5    |  2   |
|  4   |  4    |  5   |
|  5   |  5    |  4   |
